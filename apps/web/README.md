# @flying-leads/web

Frontend Next.js do Flying Leads Generator (dashboard, fila de aprovação, CRM). Ver [`ARCHITECTURE.md`](../../ARCHITECTURE.md) na raiz do repositório.

## Estado atual

Shell mínimo (Fase 0 do [`ROADMAP.md`](../../ROADMAP.md)): página única placeholder, Tailwind configurado, ainda sem shadcn/ui inicializado nem features de produto.
